# shell-class

[shell-class](https://github.com/itsall-code/shell-class) 
一个用Shell 写的课程设计

## 运行环境:<br>
1. bash
2. Fedora

## 运行方法
1. 克隆该文件或下载该压缩包<br>
```shell
git clone git@github.com:itsall-code/shell-class.git
```

2. 访问该文件夹或解压后访问<br>
```shell
cd ./shell-class
```

3. 提供运行权限<br>
```shell
chmod +x ./demo.sh
```

4. 运行<br>
```shell
bash ./demo.sh
```
